# HR Analytics – Employee Attrition Dashboard

## Overview
This project analyzes employee attrition using Power BI to identify key workforce trends and retention challenges.

## Tools Used
- Power BI
- DAX
- Excel

## Key Metrics
- Total Employees
- Attrition Count
- Attrition Rate (%)
- Average Monthly Salary

## Analysis Performed
- Attrition by Department
- Attrition by Age Group
- Attrition by Salary Level
- Attrition by Years at Company
- Interactive filtering using slicers (Department, Gender, Job Role)

## Key Insights
- Highest attrition observed in Research & Development and Sales departments.
- Employees aged 25–35 show the highest attrition.
- Lower salary bands contribute most to employee attrition.
- Majority of attrition occurs within the first few years of employment.
